My Professional Journey

In my career I have worked many roles across the major components of a software application.

Read my story

Northern Trust Asset Servicing

While in this role I have had the opportunity to work with many tools, initially I worked in more of a DevOps role. I developed a few YAML pipelines across the DEV and QA environment that created and pushed a docker image to an Azure WebApp. I also was exposed to terraform at this time. Later I began working more as a backend engineer, developing Spring services that interact with a blockchain network. I also work with the blockchain.

Northern Trust Asset Management

As a data engineer in this role I worked quite a lot with Snowflake, a cloud-based data warehouse. In this role I developed numerous ETL processes, views and tables. My scrum team was primarily focused on delivering data to client reports. Some of the views I created had quite complex calculations that needed to be done, like having nested window functions. I also worked with Spring microservices in this role as well, with the main purpose of sending JSON formatted data.

Computer Science Teaching Assistant

I worked as a teaching assistant for multiple courses, varying from intro to advanced level courses. For my introductory courses, I would write Java test cases to automate grading. I also was responsible for leading labs where I would explain core Computer Science courses. In my more advanced courses I would grade more complex assignments that where written across several coding languages. For one course, I would grade C++ and Assembly assignments.

My Computer Science Degree

While studying Physics and Computer Science I got to take several very interesting courses. With my Physics minor, I got to take a research class that gave me a real appreciation for data processing and data analytics. In my junior year I took Programming Languages, we would code in a different language every two weeks which was so much fun. However, my two favorite courses were Software Development, and Software Engineering. Each course, we were placed randomly in a 3 person team and asked to develop a product in 8 weeks.
