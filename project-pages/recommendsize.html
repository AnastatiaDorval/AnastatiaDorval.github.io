RecommendSize

A shopping tool to find the right fit.

Source Code

RecommendSize: The Tech Stack

This project is primarily written in Javascript, and as a chrome extension it uses the chrome extension API. The primary purpose of this extension is to recommend a size. It saves user entered info related to their respective size in the browser, then compares that data to the size guide info it scrapes from the clothing site. After comparing the sizeguide with the user measurements, the application displays a size recommendation.

RecommendSize: Inspiration

The inspiration behind this application started with my frustration with online shopping and inconsistent sizing systems. Women's clothing in particular is very inconsistent across brands and in order to find your actual size, you have to refer to the size guide. I wanted to create a tool that would automate that process and sort of eliminate the stress and shame-inducing experience of having to measure one's self over and over just to find their actual size.
